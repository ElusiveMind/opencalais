RAP-RDF

INTRO
=====

This module integrates ARC2, a solid PHP implementation of RDF, into Drupal.
http://arc.semsol.org

INSTALLATION
============

1) Download ARC2 from: 
http://arc.semsol.org/

Unpack and place under the modules directory so that the library's root
folder's is in modules/arc_rdf/arc2. The "modules" is the modules
folder where arc_rdf Drupal module is installed and can be:
modules, sites/all/modules or sites/default/modules.

This module was tested with 2008-03-20 release of ARC2

2) Enable the arc-rdf  module in Drupal, at:
   administration -> site configuration -> modules (admin/build/modules)
